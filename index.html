Política de Privacidade

Esta política explica como coletamos, usamos e protegemos os dados dos usuários.

Coleta de Dados

Podemos coletar informações como nome, e-mail e dados de navegação para melhorar nossos serviços.

Uso das Informações

Os dados coletados são usados para análise, melhoria do serviço e comunicação com os usuários.

Proteção

Adotamos medidas para proteger os dados contra acessos não autorizados.

Contato
